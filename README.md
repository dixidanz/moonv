# MoonV

Vue 3 + TypeScript + Tailwind CSS

### 功能

- **首頁推薦** - 熱門電影、評分最高電影、電影合集展示
- **電影搜尋** - 快速搜尋任何電影
- **電影分類** - 按類型瀏覽電影
- **電影詳情** - 查看電影詳細信息、演員陣容、預告片
- **深色模式** - 支持明暗主題切換

### API
Data provided by The Movie Database (TMDb)
This product uses the TMDB API but is not endorsed or certified by TMDB.
